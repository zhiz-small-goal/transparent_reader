TransparentMdReader 使用说明

TransparentMdReader 是一款透明的 Markdown 阅读器，可置顶、无边框，支持记忆阅读进度和最近文件。

安装与启动

  1. 解压应用包，保持目录结构不变。
  2. 确认 resources/web 目录下有 index.html、main.js、style.css、markdown-it.min.js。
  3. 将图标放在 resources/icons/app.ico（任务栏与托盘共用）。
  4. 运行 TransparentMdReader.exe。

基础操作

  • 打开文件：按 Ctrl+O，或将 Markdown 文件拖拽到窗口；托盘菜单也可打开。
  • 阅读控制：鼠标滚轮或右键翻页（锁定状态下可配置）；自动记忆滚动位置。
  • 锁定/解锁：点击标题栏锁图标；按住 Ctrl 临时解锁。
  • 托盘：单击/双击托盘图标切换显示/隐藏；最近列表可快速打开。

设置

阅读样式：字号、字体颜色/透明度、背景透明度、代码块透明度、滚动条显示。

窗口：自定义宽高（有最小值限制）。

历史与最近：调整历史记录和最近文件上限。

常见问题

  • 打开后空白或样式异常：检查 resources/web 是否完整。
  • 任务栏/托盘无图标：确认 resources/icons/app.ico 存在。
  • 无法滚动或翻页：检查是否处于锁定状态。
